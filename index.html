My Skills

ini beberapa skill yang saya kembangkan di kelas 11 semester 1

HTML

HTML, singkatan dari HyperText Markup Language, adalah bahasa standar yang digunakan untuk membuat struktur dasar dan konten sebuah halaman web

CSS

CSS, singkatan dari Cascading Style Sheets, adalah bahasa yang digunakan untuk mengatur tampilan visual dan gaya elemen di halaman web yang ditulis dalam bahasa markup seperti HTML

JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web interaktif dan dinamis, bekerja bersama HTML (struktur) dan CSS (gaya)
